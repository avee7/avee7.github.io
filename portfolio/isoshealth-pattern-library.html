Isoshealth Pattern Library

UX design UI design UI Development

In 2017, during my role as a Digital Designer at Activate Media, a startup incubator, I identified the imperative for enhancing and expediting our development processes. To address this challenge, I recommended the implementation of a Living Pattern Library.

This proposal aimed to facilitate more effective communication between our design and development teams and accelerate the design and development workflows.

Project

As the sole designer on the team, I assumed responsibility for establishing foundational styles and UI components within Sketch. Drawing inspiration from ReactJS Storybook, I created a dynamic living pattern library using ReactJS. This inventive approach empowered developers to efficiently construct user interfaces, resulting in significant time and effort savings.

The library can be found here

Some pages from the living pattern library

One of the most impressive things about the living pattern library was that it updated itself automatically. When we made changes to styles or components in the main application, those changes instantly appeared in the pattern library. This made the pattern library the ultimate source of truth, and we didn't have to spend extra time keeping it up to date.
